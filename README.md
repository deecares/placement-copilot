# 🚀 Placement Copilot

Placement Copilot is an AI-powered career assistance platform that helps students prepare for placements by analyzing resumes, evaluating ATS compatibility, extracting technical skills, and providing personalized career recommendations. Built with a modern full-stack architecture, it combines an interactive frontend with AI-powered backend services to deliver actionable resume insights.

---

## 🌟 Features

- 📄 AI-powered Resume Analysis
- 🎯 ATS Compatibility Score
- 🛠️ Automatic Skill Extraction
- 💼 Personalized Job Role Recommendations
- 📈 Resume Improvement Suggestions
- 📂 PDF Resume Upload
- ⚡ Fast and Responsive User Interface
- 🌐 Fully Deployed Full-Stack Application

## 🚀 Features in Detail

### 📄 Resume Upload

Upload resumes in PDF format for AI-powered analysis.

### 📊 ATS Score

Evaluates resumes based on Applicant Tracking System (ATS) standards and provides an ATS compatibility score.

### 🧠 Skill Extraction

Automatically identifies technical skills and keywords present in the uploaded resume.

### 💼 Career Recommendations

Suggests suitable job roles based on detected skills and resume content.

### ✨ Resume Feedback

Provides personalized suggestions to improve resume quality and increase interview readiness.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/deecares/placement-copilot.git
cd placement-copilot
```

### Backend Setup

```bash
cd backend

pip install -r requirements.txt

uvicorn app:app --reload
```

### Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

## 🎯 How It Works

1. Upload a resume in PDF format.
2. Resume text is extracted by the backend.
3. Google Gemini analyzes the resume.
4. Technical skills are identified.
5. ATS compatibility is evaluated.
6. Personalized feedback and job role recommendations are displayed.
